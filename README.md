# Health Care Management System

A modern Health Care Management System website built using Lovable and TanStack Start. It includes doctor listing, appointment booking, user interface, and healthcare service management features.

## Features
- Home Page
- Doctor Listing
- Patient Appointment Booking
- Contact Page
- Admin/Management Features
- Responsive Healthcare UI

## Technologies Used
- Lovable
- TanStack Start
- TypeScript
- React
- Vite
- Supabase
- CSS / Tailwind CSS
- GitHub
